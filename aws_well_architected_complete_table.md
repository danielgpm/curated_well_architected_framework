## Framework Overview

The AWS Well-Architected Framework is built on six pillars:

1. **Operational Excellence** - Design, delivery and operation of workloads in the cloud
2. **Security** - Protecting information, systems, and assets while delivering business value
3. **Reliability** - Ensuring a workload performs its intended function correctly and consistently
4. **Performance Efficiency** - Using computing resources efficiently to meet requirements
5. **Cost Optimization** - Avoiding unnecessary costs and delivering business value at the lowest price point
6. **Sustainability** - Minimizing the environmental impacts of running cloud workloads
