## Framework Overview

The AWS Well-Architected Framework helps cloud architects build secure, high-performing,
resilient, and efficient infrastructure for a variety of applications and workloads.

### The Six Pillars

1. **Operational Excellence** - Design, delivery and operation of workloads in the cloud
2. **Security** - Protecting information, systems, and assets
3. **Reliability** - Workload performing its intended function correctly and consistently
4. **Performance Efficiency** - Using computing resources efficiently
5. **Cost Optimization** - Delivering business value at lowest price point
6. **Sustainability** - Minimizing environmental impacts of running cloud workloads
